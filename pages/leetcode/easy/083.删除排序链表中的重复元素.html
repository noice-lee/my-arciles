# 删除排序链表中的重复元素

# 题目

给定一个已排序的链表的头 head ， 删除所有重复的元素，使每个元素只出现一次 。返回 已排序的链表 。

示例 1:

输入：head = [1,1,2]
输出：[1,2]
1
2

示例 2:

输入：head = [1,1,2,3,3]
输出：[1,2,3]
1
2

提示：

  • 链表中节点数目在范围 [0, 300] 内
  • -100 <= Node.val <= 100
  • 题目数据保证链表已经按升序 排列

# 思路

循环链表，如果当前节点值与下个节点值相等则下个节点指向下下个

# 代码

function deleteDuplicates(head: ListNode | null): ListNode | null {
  let i: ListNode | null = head;
  while (i && i.next) {
    if (i.val === i.next.val) {
      i.next = i.next.next;
    } else {
      i = i.next;
    }
  }
  return head;
}
