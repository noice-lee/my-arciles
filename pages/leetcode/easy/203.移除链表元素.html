# 移除链表元素

# 题目

给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。

示例 1：

输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
1
2

示例 2：

输入：head = [], val = 1
输出：[]
1
2

示例 3：

输入：head = [7,7,7,7], val = 7
输出：[]
1
2

提示：

  • 列表中的节点数目在范围 [0, 104] 内
  • 1 <= Node.val <= 50
  • 0 <= val <= 50

# 思路

首先添加哨兵元素，可让循环判断更加方便，循环判断链表，如果有指定的整数，则将next指向下一个，循环结束后返回链表

# 代码

var removeElements = function (head, val) {
  // 添加哨兵元素
  let ele = {
    next: head,
  };
  let p = ele;
  while (p.next) {
    if (p.next.val === val) {
      p.next = p.next.next;
    } else {
      p = p.next;
    }
  }
  return ele.next;
};
